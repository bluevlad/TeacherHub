# TeacherHub E2E 테스트

Playwright 기반 E2E(End-to-End) 테스트 환경입니다.

## 테스트 환경

| 항목 | URL |
|------|-----|
| Frontend | http://study.unmong.com:4010 |
| API Server | http://study.unmong.com:8081 |

## 설치

```bash
cd e2e
npm install
npx playwright install chromium
```

## 테스트 실행

```bash
# 전체 테스트 실행
npm test

# UI 모드로 실행 (디버깅에 유용)
npm run test:ui

# 브라우저 표시하며 실행
npm run test:headed

# 디버그 모드
npm run test:debug

# 테스트 결과 리포트 보기
npm run test:report

# 특정 프로젝트만 실행
npm run test:chromium
npm run test:mobile
```
