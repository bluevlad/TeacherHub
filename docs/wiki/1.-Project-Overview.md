# 1. 프로젝트 개요 (Project Overview)

## 1.1 프로젝트 소개

### 프로젝트 정보
| 항목 | 내용 |
|------|------|
| **프로젝트명** | TeacherHub |
| **버전** | 1.0.0 |
| **시작일** | 2024년 |
| **상태** | Active Development |
| **라이선스** | MIT |

### 한줄 요약
> 공무원 학원 강사에 대한 온라인 평판을 수집하고 분석하여 일별/주별/월별 트렌드를 제공하는 감성 분석 플랫폼

### 상세 설명
TeacherHub는 다음과 같은 가치를 제공합니다:

1. **데이터 수집**: 디시인사이드 공무원 갤러리에서 강사 관련 게시글 자동 수집
2. **감성 분석**: 게시글 내용의 긍정/부정/중립 감정 자동 분류
3. **기간별 통계**: 일별, 주별(ISO 8601), 월별 집계 데이터 제공
4. **트렌드 파악**: 시간에 따른 강사별 평판 변화 추적

## 1.2 비전 및 목표

### 비전 (Vision)
```
"온라인에 흩어진 강사 평판 정보를 체계적으로 수집하고 분석하여
수험생들이 더 나은 선택을 할 수 있도록 돕는다"
```

### 미션 (Mission)
- 강사 관련 언급의 **자동 수집**
- **객관적인** 감성 분석 제공
- **시계열** 트렌드 분석
- **투명한** 데이터 제공

### 핵심 목표
| 목표 | 측정 지표 | 목표치 |
|------|----------|--------|
| 데이터 커버리지 | 일일 수집 게시글 | 100+ |
| 분석 정확도 | 감성 분류 정확도 | 80%+ |
| 시스템 가용성 | 업타임 | 99%+ |
| 데이터 신선도 | 수집 지연 시간 | 24시간 이내 |

## 1.3 핵심 가치 제안 (Value Proposition)

### 수험생

```
┌─────────────────────────────────────────────────────────┐
│                     Before                               │
├─────────────────────────────────────────────────────────┤
│ • 여러 커뮤니티를 직접 검색해야 함                       │
│ • 정보가 파편화되어 있음                                 │
│ • 최신 정보인지 확인 어려움                              │
│ • 주관적인 의견과 객관적 평가 구분 어려움                │
└─────────────────────────────────────────────────────────┘
                           ▼
┌─────────────────────────────────────────────────────────┐
│                   After (TeacherHub)                     │
├─────────────────────────────────────────────────────────┤
│ ✓ 한 곳에서 모든 강사 평판 확인                          │
│ ✓ 일별/주별/월별 트렌드 파악                             │
│ ✓ 감성 분석으로 객관적 지표 제공                         │
│ ✓ 랭킹 시스템으로 비교 용이                              │
└─────────────────────────────────────────────────────────┘
```

## 1.7 프로젝트 범위

### In Scope (포함)
- 디시인사이드 공무원 갤러리 크롤링
- 등록된 강사 기반 언급 탐지
- TextBlob 기반 감성 분석
- 일별/주별/월별 통계 대시보드
- Docker 기반 배포

### Out of Scope (미포함)
- 다른 커뮤니티 크롤링 (향후 확장 가능)
- 실시간 알림 시스템
- 사용자 인증/로그인
- 강사 직접 등록 기능
